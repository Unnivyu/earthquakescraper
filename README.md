# PhilvolcsEarthquakeScraper

This project provides a web scraping script that collects the monthly recorded earthquakes from PHIVOLCS (Philippine Institute of Volcanology and Seismology) and exports the data into a CSV file.

---

## Requirements

Install dependencies using the provided `requirements.txt`:

```bash
pip install -r requirements.txt
```
---
Or manually install the required packages:
```bash
pip install beautifulsoup4 pandas requests
```

### How to run
```bash
python erthscraper.py
```
### Output:
A CSV file named: earthquake.csv will be generated within the current directory
